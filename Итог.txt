Для выполнения задания выбран датасет "Загрязнение воздуха в Китае (2015-2025)".

В качестве целевой переменной выбрана AQI - индекс качества воздуха.

Матрица корреляции данных с AQI отражает слабую свзять данных датасета целевой переменной
значения находятся в диапазоне от -0,07 до 0,03).

Для регрессии были выьраны следующие алгоритмы:
LinearRegression
Gradient Boosting Regressor
Light Gradient Boosting Machine Regressor
Random Forest Regressor
ExtraTreesRegressor
Ансамбль регрессоров(LinearRegression, DecisionTreeRegressor, RandomForestRegressor, GradientBoostingRegressor).

В препроцессинге произведено масштабирование, разделение данных на числовые и категориальные, а также 
исключены неинформативные признаки (широта, долгота - есть город, дата фиксации измерения - есть сезоны, 
ID станции).

Вместе с тем, по результатм работы алгоритмов можно отметить довольно низкую метрику на тестовых данных 
(MAE, MSE, RMSE, R2).
Дополнительно была снята метрика на обучающем наборе. В целом, метрики на обучающем наборе и на тестовом 
наборе являются небольшими, что может говорить о том, что модель не переобучается.
Минимальное значение R2 достигнуто ExtraTreesRegressor = -0,0029 (с учетом подобранных лучших параметров:
{'max_depth': 1, 'max_features': 1, 'min_samples_leaf': 5, 'min_samples_split': 10, 'n_estimators': 95}).

LinearRegression подтверждает отсутствие линейной зависимости в данных.

Дополнительно стоит отметить, что алгоритмами было осуществлено разное ранжирование признаков важности, 
хотя расчет AQI включает измерение концентраций загрязнителей (например, PM2.5, PM10, O₃, CO, SO₂, NO₂) 
и использование интерполяционной формулы, которая преобразует эти концентрации в значение на универсальной 
шкале от 0 до 500.
